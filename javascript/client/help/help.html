In the chat windows you can specify you are expressing a command by putting a '#' in front of what you write.
every valid expression is build like so:
'#Verb Noun a', while
Verb is replaced by Set or Get and
Noun is replaced by Round and
a is replaced by a natural number smaller than the maximum amount of playable rounds by the given amount of players and bigger than zero
for example '#Set Rounds' with Verb = 'Set' and Noun = 'Rounds'
causes the round after which the game ends to be set to the number you enter seperated by a space after '#Set Rounds'
so for example '#Set Rounds 6' sets the game to end after the sixth round.

possible verbs = { Set, Get }
possible nouns = { Rounds }
